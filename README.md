# 流浪图床上传器（适用于Typora）

用于上传图片到[流浪图床](https://p.sda1.dev/)。

可以搭配Typora的上传一起使用



## 使用

- **必要条件：安装python**



- 命令行上传（可单张或多张）：

```
python {存放路径}\main.py {图片1} {图片2} {图片3}
```

- Typora自定义上传：

```
python {存放路径}\main.py ${filename}
```
